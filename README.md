# 📊 Exploratory Data Analysis (EDA) – Titanic Dataset

## 🔹 Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns, relationships, and key factors influencing passenger survival.

---

## 🎯 Objectives
- Clean and preprocess raw data  
- Perform statistical analysis  
- Visualize data using graphs  
- Identify key factors affecting survival  

---

## 📁 Dataset
- Source: Kaggle – Titanic Dataset  
- Target Variable: **Survived (0 = No, 1 = Yes)**  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🔄 Workflow
1. Data Cleaning (handling missing values, dropping columns)  
2. Univariate Analysis (distribution of features)  
3. Bivariate Analysis (relationships between variables)  
4. Correlation Analysis (heatmap)  
5. Multivariate Analysis (pairplots)  

---

## 📊 Key Insights
- Females had higher survival rates than males  
- First-class passengers survived more  
- Age plays a role in survival probability  
- Fare and passenger class are correlated
